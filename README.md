Welcome to my repository! This workspace contains the tasks and projects I completed during my Python Programming Internship with **CodeAlpha** (May 2026). 

As a software engineering student, my goal in this internship is to build logic from scratch, understand control flow deeply, and write clean, maintainable code without relying on pre-built templates.
