# Task 1: Hangman Game 🎮

A classic, text-based Hangman game played directly in the terminal.

## 📝 Description
The objective of this task was to create a word-guessing game. Instead of simply masking single words, this version includes advanced logic to handle multi-word phrases (displaying spaces correctly) and robust input validation to prevent crashes from invalid user entries.

## ✨ Features
* **Randomized Selection:** Picks a random phrase from a predefined custom list.
* **Input Validation:** Ensures the user only enters single, alphabetical characters.
* **Smart Formatting:** Automatically reveals spaces between words.
* **Attempt Tracking:** Keeps track of remaining guesses (max 6) and previously guessed letters.

## 🚀 How to Run
1. Open your terminal.
2. Navigate to this directory.
3. Run the following command:
   ```bash
   python hangman.py
